# Dashboard Project

A beginner-level dashboard project built using HTML, CSS, and JavaScript.

This project includes UI and UX features such as dark mode and light mode.  
The dashboard shows analytics, reports, and provides advice messages when data changes.

## Features
- Dark mode and light mode toggle
- Default sales analytics
- Ability to increase or decrease sales value
- Dynamic messages that appear when sales decrease
- Dashboard report cards and analytics display
- Responsive UI

## Technologies
- HTML
- CSS
- JavaScript
